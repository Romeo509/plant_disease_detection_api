API Endpoint

POST /analyze

Upload plant images to get condition analysis and treatment recommendations.

Request Format

Content-Type: multipart/form-data

Parameter: media - Plant images (JPG, PNG)

Example Request

curl -X POST /analyze \
  -F "media=@plant_image.jpg" \
  -H "Accept: application/json"

Response Format

Success Response

Status: 200 OK

{
  "success": true,
  "analysis": {
    "plant_type": "Tomato",
    "condition_status": "Early Blight",
    "control_treatment": "Apply copper fungicide"
  }
}

Error Response

Status: 400 Bad Request

{
  "error": "No media files uploaded"
}
